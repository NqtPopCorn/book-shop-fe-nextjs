# Book Shop Frontend

Frontend mới migrate sang Next.js App Router, TypeScript, Zustand, TanStack React Query, Axios, Tailwind CSS và shadcn/ui.

## Chạy local

```bash
cp .env.example .env.local
npm install
npm run dev
```

Mở http://localhost:3000

API mặc định: `http://localhost:3000/api`

Không sao chép `.env` từ repository cũ; chỉ dùng `.env.example`.
